Git [Version Control]
 - DVCS (Distributed Version Control System)
 
[Command Line]
--HELP
$ git help
$ git help <command>

--SETUP
$ git config --global user.name "Luis Garcia"
$ git config --global user.email "luisgarcia173@gmail.com"
$ git config --global color.ui true

--CREATE REPO
$ mkdir <folder>
$ cd <folder>
$ git init

--STATUS [TRACKING]
$ git status
